## 📝 備考

### 技術的な注意点

1. **メールアドレスの変更**
   - `.env.local` はGit管理外なので、デプロイ時に手動設定が必要
   - 本番環境では `RESEND_API_KEY` も更新すること

2. **見出しサイズの拡大**
   - `tracking-tight2` クラスで字詰めを最適化済み
   - `leading-tight` で行間を調整済み
   - さらに大きくする場合は `line-height` も調整が必要

3. **画像の配置**
   - Next.js Image コンポーネントの `priority` 属性を使用してLCPを最適化
   - `sizes` 属性でレスポンシブ対応
   - WebP 形式を推奨（PNG からの自動変換が可能）

### デザイン上の注意点

1. **イラストの統一性**
   - カラーパレットを統一（#2563eb ブランドブルー）
   - スタイルを統一（フラット、アイソメトリック）
   - 人物の顔は避ける

2. **見出しの視覚階層**
   - H1: 最も目立つ（キャッチコピー）
   - H2: セクション見出し
   - H3: サブセクション見出し
   - 本文との対比を保つ

## 🆘 トラブルシューティング

### メールが届かない場合
1. `.env.local` の `ADMIN_EMAIL` を確認
2. Resend のダッシュボードでログを確認
3. スパムフォルダを確認

### 見出しが大きすぎる場合
1. `app/globals.css` の `.h1`, `.h2`, `.h3` を調整
2. `text-{size}` の値を1段階下げる（例: 7xl → 6xl）

### 画像が表示されない場合
1. `/public/images/` に配置されているか確認
2. ファイル名が正しいか確認
3. Next.js Image の `src` パスが `/images/...` になっているか確認
